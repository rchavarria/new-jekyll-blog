# Run Jekyll locally with docker

Install ruby bundles:

```bash
docker run \
  -v .:/srv/jekyll \
  -v ./vendor/bundle:/usr/local/bundle \
  --name local-jekyll \
  jekyll/jekyll \
  bundle install
```

Run with docker compose:

```bash
cd docker
docker-compose up &
```
